### MindScribe 321
MindScribe 321 is a auto-mindmapping software. It allows users to take notes in a docs note feild and have a mind map automatically created as they work. 
### First Prototype
<img width="1919" height="890" alt="Prototype" src="https://github.com/user-attachments/assets/a6b38e7c-81be-4b18-8812-86e3566c0a97" />
Currently the only functionality is extracting header text and printing it. Uses the tinymce api.
### First sprint
<img width="1381" height="804" alt="Screenshot 2026-04-09 at 2 32 39 PM" src="https://github.com/user-attachments/assets/70c6c9c0-8d85-4c3a-8b78-3b5fe1b14267" />
A solid versions of Feature 1 is implemented with a graph being automatically generated depending on the headers. It turns header text on the left into ellipse nodes on the right that are then structured based on the header hierarchy. The ellipse nodes also dynamicaly resize depending on the text width. There is still an issue of the nodes becoming too long and overlapping each other.
